# Security Policy

## Supported Versions

Only the latest released versions of microstack-dev packages are supported with
security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Do not open a public issue or disclose the vulnerability publicly.

Provide:
- A clear description of the issue
- Steps to reproduce (if possible)
- Affected package and version

## Response

We aim to acknowledge reports within a reasonable timeframe and will coordinate
a fix before any public disclosure.

Thank you for helping keep microstack-dev projects secure.
